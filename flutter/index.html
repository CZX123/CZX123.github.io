Installation

Windows

  1. Download zip from https://flutter.dev/docs/get-started/install/windows
  2. Extract it into any folder, preferably in your own user folder, i.e. C:\Users\(YourUsername)\
  3. Copy the path of the bin folder in the flutter folder, e.g. C:\Users\(YourUsername)\flutter\bin\
  4. Add this folder to your path
    1. Search for path in Windows Search. ‘Edit the system environment variables’ will come up. Click on it.
    2. Go to ‘Environment Variables’.
    3. Double click on ‘PATH’
    4. Click ‘New’, and paste the previous path copied (C:\Users\(YourUsername)\flutter\bin\) into the field.
  5. Open Command Prompt and just type ‘flutter’ and run it
    1. Flutter will proceed to download lots of stuff, just wait
  6. Once done, run ‘flutter doctor -v’ to see the other components needed to be installed.
  7. For simplicity sake, download the entire Android Studio. (Only the Android SDK, Android SDK Platform-Tools and Android SDK Build-Tools are required)
  8. Run flutter doctor again to check if everything is correct

MacOS / Linux

  1. Download zip from https://flutter.dev/docs/get-started/install/macos or https://flutter.dev/docs/get-started/install/linux
  2. Extract it into any folder, preferably in your own user folder, i.e. /Users/(YourUsername)/
  3. Copy the path of the bin folder in the flutter folder, e.g. /Users/(YourUsername)/flutter/bin/
  4. Add this folder to your path
  5. Open Terminal and just type ‘flutter’ and run it
    1. Flutter will proceed to download lots of stuff, just wait
  6. Once done, run ‘flutter doctor -v’ to see the other components needed to be installed.
  7. If you want to test on Android, download the entire Android Studio
  8. Download any emulators if you lack a physical Android device.
  9. For testing on iOS in MacOS, follow instructions here: https://flutter.dev/docs/get-started/install/macos#ios-setup
  10. Run flutter doctor again to check if everything is correct
